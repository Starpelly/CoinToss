# Coin Toss
Mobile port of Coin Toss from Rhythm Heaven DS

## Compilation
* Download the source code
* Install Unity 2022*
* Open the project
* Input your LootLocker keys for leaderboards to work [here](https://github.com/LootLocker/unity-sdk)
